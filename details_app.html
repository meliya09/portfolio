Project information

  • Link Github Backend: frontend_mobile_bpdsmart

Project Details

  • BPD DIY SMART is a digital-based application that is able to provide information and guidance to improve employee performance and service to customers. The application is developed based on mobile digital technology (android). The word Smart means smart/intelligent. The letters in the word Smart can be interpreted Integrated Marketing System.
  • BPD DIY SMART displays the latest data information for the internal interests of the Bank from the head office to the entire service network in real time. The information presented is an integrated Marketing Tools with a variety of information needed internal Bank, among others, news information, bank products, network services, etc.
  • Database : PostgreSQL
  • Backend : Rest API Codeigniter 4
  • Frontend Mobile : Flutter
